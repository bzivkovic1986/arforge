# ARForge Documentation

ARForge is a YAML-first AUTOSAR Classic modeling tool. You describe your project in plain text, ARForge validates it semantically, and exports standards-compliant ARXML.

## Where to start

If you are new to ARForge, read the docs in this order:

1. [Overview](./overview.md) — what ARForge does and where it fits in a workflow
2. [Project Structure](./project-structure.md) — how a project is laid out on disk
3. [Modeling Concepts](./modeling-concepts.md) — the YAML modeling language explained
4. [Validation](./validation.md) — how validation works and what the finding codes mean
5. [CLI](./cli.md) — all commands and options
